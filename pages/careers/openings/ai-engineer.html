About the role

We're hiring an AI Engineer to lead our use of modern AI development tooling — Claude Code, GitHub Copilot, Cursor, Microsoft Copilot Studio, and the rapidly evolving ecosystem of AI-powered engineering and business agents.

This role is different from our Prompt Engineer or Agent Engineer roles. Those engineers build LLM systems and agentic platforms. You build with AI tooling — using Claude Code to ship faster, designing Copilot Studio agents for client business processes, and helping our clients adopt AI tooling productively in their own organizations.

If you've been quietly 3-5x'ing your engineering output using Claude Code or Cursor for the last year, and you have opinions about which AI tools actually work versus which are hype — this is the role for you.

What you'll do

  • Use Claude Code, Copilot, and Cursor as primary tools for shipping engineering work — set the standard for what AI-assisted development looks like at AlpineQora
  • Build Microsoft Copilot Studio agents for client business workflows — when low-code is the right answer
  • Design and operationalize internal AI tooling — Claude Skills, custom MCP servers, prompt libraries, eval suites
  • Help clients adopt AI tooling — pilots, training, governance frameworks, ROI measurement
  • Build proof-of-concepts rapidly — when a client asks "can this be done?", you're the person who shows them in days, not weeks
  • Stay ahead of the tooling curve — new releases, new patterns, new failure modes
  • Partner with our Prompt Engineer and Agent Engineer on tooling that bridges low-code (Copilot Studio) and full-code (custom agents) approaches
  • Contribute to public technical writing — what works, what doesn't, what's worth waiting for
  • Help define what "AI-native engineering" looks like in practice for an enterprise services firm

What we're looking for

Must have

  • 4+ years of professional software engineering with a strong production track record
  • 1+ year of daily, hands-on work with modern AI dev tooling (Claude Code, GitHub Copilot, Cursor, or equivalent)
  • Practical experience building agents in Microsoft Copilot Studio — or strong evidence you can learn it fast (e.g., Power Platform / Power Automate background)
  • Demonstrated ability to ship working software using AI tools — concrete examples, not just claims
  • Strong fundamentals in at least one of: Java, Python, JavaScript / TypeScript, C#
  • Clear point of view on when AI tooling helps and when it doesn't — you don't suggest "ask an LLM" for problems that need a database query
  • Comfort with rapid prototyping — you can take a vague idea and produce a working POC in days
  • Excellent technical writing — your documentation, prompt libraries, and runbooks need to be usable by others
  • Healthy skepticism of AI hype balanced with genuine excitement about what does work

Nice to have

  • Microsoft 365 ecosystem expertise — Copilot, Power Platform, Power Automate, SharePoint, Teams
  • Experience with custom MCP server development (Anthropic's Model Context Protocol)
  • Background in change management or AI adoption programs
  • Familiarity with enterprise AI governance — data residency, audit, compliance
  • Experience operating in regulated industries (fintech, healthcare, government)
  • Public artifacts — open source contributions, blog posts, conference talks on AI tooling
  • Background as an engineer-of-engineers — internal tools, developer productivity, platform engineering
  • Experience with Anthropic's Claude API, OpenAI API, or other LLM provider APIs

Why join AlpineQora

  • Set the standard for AI-native engineering at a firm building this from the ground up
  • Real autonomy to choose tools and patterns — we're not a Microsoft-only shop, not a Claude-only shop, not anything-only
  • Cross-cutting impact — your work touches engineering productivity, client deliverables, and public point of view
  • Direct partnership with the Prompt Engineer and Agent Engineer roles
  • Hybrid work — 3 days office in Gandhinagar, 2 days WFH
  • Comprehensive health insurance + family coverage
  • Annual learning budget — Microsoft Build, AnthropicDev, GitHub Universe, courses, API credits for experimentation
  • Performance bonus tied to real outcomes

The process

A short, respectful process — typically 2-3 weeks from first conversation to offer:

  • Intro call (30 min) — talk about your work with AI tooling, your favorite tools, your least favorite tools
  • Live build session (90 min) — build something real together using Claude Code or Copilot Studio for a problem we give you
  • Portfolio walkthrough (60 min) — walk us through work you've shipped using AI tools, the prompts and patterns you've developed
  • Final conversation — with the founding team; talk about the future of this discipline

Notice period

Up to 60 days considered for strong candidates. Immediate joiners preferred.
